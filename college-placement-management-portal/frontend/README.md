
# Placement Portal Frontend (Vite + React + Tailwind)

## Quick start
1. Install dependencies (Node 18+ recommended):
   npm install

2. Start dev server:
   npm run dev

3. Open http://localhost:3000

## Notes
- This frontend proxies /api requests to http://localhost:5000 (configured in vite.config.js)
- Update proxy if your backend runs on a different port.
